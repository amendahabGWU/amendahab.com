Add playable audio files to this folder and point `AUDIO_PLAYLIST` in `index.html` at them.

Examples:
- `media/without-fear-of-wind.wav`
- `media/without-fear-of-wind.mp3`
- `media/interlude.wav`
- `media/finalwedding.jpg` for artwork

Supported in the player scaffold:
- audio: `.mp3`, `.wav`, `.ogg`, `.m4a`
